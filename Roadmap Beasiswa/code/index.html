Roadmap Beasiswa

roadmapbeasiswa.com adalah panduan lengkap untuk persiapan beasiswa dari SMP hingga perguruan tinggi. Temukan program beasiswa yang tepat dan sumber belajar gratis untuk meraih impian pendidikan Kamu.
Beasiswa dan Sumber Belajar

  • Beasiswa untuk SMP

    AFS Intercultural Programs

    Program pertukaran pelajar yang memberikan kesempatan kepada siswa SMP kelas 8–9 untuk mengembangkan skill interbudaya dan belajar di negara lain.

    ASEAN Scholarships

    Beasiswa dari negara-negara ASEAN untuk siswa SMP berprestasi akademik dan non-akademik, mendorong kerja sama dan pemahaman antar budaya di kawasan Asia Tenggara.

  • Beasiswa untuk SMA

    AFS Intercultural Programs

    Program pertukaran pelajar untuk siswa SMA kelas 10–11 ke 40+ negara, memperkaya pengalaman budaya dan pembelajaran global.

    Kennedy-Lugar YES

    Program pertukaran pelajar ke Amerika Serikat yang fully-funded untuk siswa SMA kelas 10–11, mempromosikan pemahaman lintas budaya dan pemimpin muda.

    ASEAN Scholarships

    Disediakan bagi siswa SMA berprestasi untuk studi di negara ASEAN, mendorong integrasi dan kerja sama di kawasan.

  • Beasiswa untuk Mahasiswa

    IISMA (Indonesian International Student Mobility Awards)

    Program Kemendikbud untuk mahasiswa S1 semester 4–7 mengikuti pertukaran di universitas mitra internasional, memperluas wawasan akademik dan global.

    LPDP (Bappenas – Kemdikbud)

    Beasiswa master/doktor untuk lulusan S1 berprestasi, mendanai studi lanjut dan pengembangan kapasitas kepemimpinan.

    Beasiswa Unggulan Kemendikbud

    Mendukung siswa berprestasi akademik/non-akademik untuk studi S1/S2, mendorong kontribusi sosial dan pembangunan nasional.

    Fulbright, DAAD, Erasmus+

    Beasiswa internasional bergengsi untuk pelamar berprestasi tinggi, memberikan kesempatan studi dan penelitian di luar negeri.

  • Sumber Belajar

    Platform gratis untuk latihan TOEFL/IELTS, panduan esai beasiswa, dan komunitas belajar online.
